# Hangman Game (Виселица)
## English
A console-based Hangman game that supports both English and Russian languages, featuring player profile management, statistics tracking, and dynamic multi-language localization.
### Features
 * **Bilingual Support:** Play the game fully in English or Russian.
 * **Player Profiles:** Saves player progress and history based on your chosen name.
 * **Score Tracking:** Tracks your lifetime total of victories and defeats.
 * **In-Game Settings:** Change your player name or preferred language directly from the game menu at any time.
### How to Play
 1. **Choose Your Language:** On the first launch, select your preferred language (English or Russian).
 2. **Enter Your Name:** Set up your profile name to keep track of your stats.
 3. **Navigate the Main Menu:** Type one of the available commands: Game, Rules, Highscores, Settings, or Exit.
 4. **Guess the Word:** In Game mode, enter one letter at a time to guess the hidden word. You have 10 incorrect attempts before the game ends!
## Русский
Консольная версия классической игры "Виселица" с поддержкой английского и русского языков, возможностью создания профилей игроков, отслеживанием статистики и динамической локализацией.
### Особенности
 * **Двуязычный интерфейс:** Полная поддержка игрового процесса на английском и русском языках.
 * **Профили игроков:** Сохранение прогресса и истории игр под вашим уникальным именем.
 * **Отслеживание статистики:** Ведение учета общего количества побед и поражений для каждого профиля.
 * **Игровые настройки:** Возможность в любой момент изменить имя игрока или язык интерфейса прямо из главного меню.
### Игровой процесс
 1. **Выбор языка:** При первом запуске выберите предпочитаемый язык (English или Русский).
 2. **Ввод имени:** Укажите имя своего профиля для сохранения личной статистики.
 3. **Главное меню:** Используйте доступные команды для навигации: Game (Игра), Rules (Правила), Highscores (Рекорды), Settings (Настройки) или Exit (Выход).
 4. **Угадывание слова:** В режиме игры вводите по одной букве, чтобы отгадать скрытое слово. У вас есть 10 попыток на ошибку, прежде чем игра завершится!
